# NaiHeQiao
初始版本 测试已过GPK 按照其工作原理及特性 其它游戏保护也是通杀的



# 如何使用？
复制 NaiHeQiao.dp32和nhq_core_x86.dll到x32/plugins目录下

复制 NaiHeQiao.dp64和nhq_core_x64.dll到x64/plugins目录下

菜单 -> 插件 -> NaiHeQiao -> enjoy it -> inject dll

 - 断点 -> 回车messagebox进行单步调试
